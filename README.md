# debug.css
> Outline your components for easy CSS debugging

## 🌈 Different versions

### All elements
All elements will be outlined by a red border
![](./screenshots/allElements.png)

### Only divs
All divs on the page will be outlined by a red border
![](./screenshots/divs.png)
